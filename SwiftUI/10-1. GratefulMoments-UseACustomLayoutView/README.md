# Grateful Moments: Use a custom layout view

Continue the Grateful Moments demo app by displaying entries in a list and building a custom hexagon layout to show each entry’s details.

In the next chapter, you’ll make the app more engaging by adding badges people can unlock.

## Overview

These resources are associated with the [Use a custom layout view](https://developer.apple.com/tutorials/develop-in-swift/use-a-custom-layout-view) tutorial. This is part of the [App Development](https://developer.apple.com/tutorials/develop-in-swift/welcome-to-app-development) tutorials from [Develop in Swift](https://developer.apple.com/tutorials/develop-in-swift) by Apple.

## Using these resources

Follow the tutorial instructions in section one of [Use a custom layout view](https://developer.apple.com/tutorials/develop-in-swift/use-a-custom-layout-view) to use these resources within the Xcode app.
